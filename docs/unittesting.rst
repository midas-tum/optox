Unittesting
===========

Python Wrappers
***************

To run all unittests for ``optopy`` use

.. code-block::

    python -m unittest discover optopy.test

To run a single unittest for, e.g., *nabla*, use

.. code-block::

    python -m unittest optopy.test.test_nabla
